## 커밋 유형

| 태그 이름    | 설명                                    |
|----------|---------------------------------------|
| Feat     | 새로운 기능을 추가할 경우                        |
| Fix      | 버그를 고친 경우                             |
| Design   | CSS, 사용자 UI 수정                        |
| Style    | 코드 포맷, lint, 코드 수정이 없는 경우             |
| Refactor | 기능 변경 없이 코드 구조 수정                     |
| Test     | 테스트 코드 작성 및 수정할 경우                    |
| Comment  | 주석 추가 및 변경                      |
| Remove   | 쓸 데 없는 파일 및 코드 블럭 삭제                  |
| Edit     | 기타 설정 변경이나 외부 api 연동 설정이 변경, 문구 수정 등등 |
